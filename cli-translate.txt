Creating Virtual Machines

Overview
In this lab, we will explore the Virtual Machine instance options and create several VMs with different characteristics.

Task 1: Create a utility virtual machine

gcloud compute instances create my-cli-vm1 --zone us-central1-c --machine-type n1-standard-1 --subnet default --image debian-9-stretch-v20200805 --image-project debian-cloud --no-address 

Task 2: Create a Windows virtual machine

** Creating first a firewall rule to allow http and https traffic

gcloud compute firewall-rules create allow-http-https --direction=INGRESS --action=ALLOW --priority=1000 --network=default --rules=tcp:80, tcp:443 --source-ranges=0.0.0.0/0 --target-tags=allow-http-s


gcloud compute instances create my-cli-vm2 --zone europe-west2-a --machine-type n1-standard-2 --image windows-server-2016-dc-core-v20200813 --image-project windows-cloud --boot-disk-size 100GB --boot-disk-type pd-ssd --tags allow-http-s
 


Task 3: Create a custom virtual machine


gcloud compute instances create my-cli-vm3 --zone us-west1-b --subnet default --custom-cpu 6 --custom-memory 32GB --image debian-9-stretch-v20200805 --image-project debian-cloud
